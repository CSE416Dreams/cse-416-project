Redistricting Plan Map Tool

What is it

Every state has a number of congressional districts, proportional to the state population. Population is recalculated after a US Census. District boundaries must be recalculated if the number of representatives change or population shifts.

How to use it

Start by navigating to the top-right - Select a state - and select a state. You can see and compare the most recent districting plans and view the statistical analysis comparing various plans.
